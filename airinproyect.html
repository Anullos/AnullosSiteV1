AirinProyect

Aplicación desarrollada en Flutter con servidor propio en Nodejs.

Es una aplicación desarrollada para los profesores donde podrán gestionar sus clases, mandar tareas, recibir mensajes de los alumnos y muchas cosas más. Cuenta con autentificación a traves de tokens para la conexión, con base de datos NoSQL de MongoDB Atlas. Además creé un servidor Nodejs desde 0 para sus API.
